아래 링크를 통해 Reliv를 다운로드 받을 수 있습니다.

[Reliv 다운로드 링크](https://drive.google.com/drive/folders/1I9wAxia6GupGkbKsbHPmcEO6HBrI4T6U)

## 1. Windows 설치

Windows 설치파일(.exe)을 다운로드 받아 설치합니다.

현재 베타 버전 빌드로, Windows 방화벽에 의해 보안 안내창이 뜰 수 있습니다.

이 때, **\[추가 정보\]** 를 클릭하여 설치를 진행합니다.

<img className="block" src="/assets/getting-started/windows-install.png" />

## 2. macOS 설치

macOS 설치파일(.dmg)을 다운로드 받아 설치합니다.

사용자의 CPU 아키텍처에 맞는 설치파일을 다운로드 받아 설치합니다.

(Intel CPU -> x64 / M1 -> arm64)
