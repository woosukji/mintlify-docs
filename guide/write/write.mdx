## 1. 기록 모드로 액션 기록하기

테스트를 생성하신 후, 시작 URL에 접속된 브라우저와 **Reliv 에디터**가 실행됩니다.

Reliv 에디터 하단의 **\[Record\]** 버튼을 눌러 기록 모드를 실행할 수 있습니다.

<img className="block" src="/assets/write/write/browser-editor.png" />

## 2. 기록 모드 종료하기

기록 모드 상태로 브라우저에서 테스트를 수행하면, 각 스텝이 에디터에 기록됩니다.

테스트 기록이 끝나면, 에디터 하단의 **\[Stop Recording\]** 버튼을 눌러 기록 모드를 종료합니다.

<Note>
  Reliv 에디터 우상단의 **\[ 💾 \]** 버튼을 눌러 작성한 테스트를 저장할 수
  있습니다.
</Note>

## 3. 테스트 재생하기

기록 모드를 종료한 후, 에디터 하단의 **\[Play All\]** 버튼을 눌러 테스트를 처음부터 끝까지 재생합니다.

<Note>
  에디터의 **\[Test Stpes\]** 패널 중 특정 액션에 마우스를 올렸을 때 나타나는
  **\[▶︎\]** (재생) 버튼을 눌러, <br />
  **현재 실행 중단점부터 해당 액션까지 테스트를 부분 재생**할 수 있습니다.
</Note>
