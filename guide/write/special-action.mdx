title: "특수 액션 추가하기"

---

---

Reliv 에디터 **\[Test Steps\]** 패널 우상단의 **\[+\]** 버튼을 눌러 특수 액션을 추가할 수 있습니다.

특수 액션의 종류는 다음과 같습니다.

- Check 액션 : 현재 상태가 기대한 상태와 같은지 확인하는 Assertion 입니다.
  - `isVisible` : 특정 요소가 화면에 표시되는지 확인합니다. <br />
    추가 시, 브라우저 화면에서 대상 요소를 선택하여 액션 추가를 완료합니다.
  - `isEqualTo` : 특정 요소의 현재 텍스트가 일치하는지 확인합니다. <br />
    추가 시, 브라우저 화면에서 대상 요소를 선택하여 액션 추가를 완료합니다.
- Add 액션
  - `Custom Action` : 직접 코드를 작성하여 [커스텀 액션](/guide/advanced/custom-action)을 정의할 수 있습니다.
  - `Action Block` : 자주 사용하는 스텝들을 묶은 [액션블록](/guide/action-block)을 추가합니다.
  - `Browser Action` : 브라우저 액션입니다.
    - `Click` : 특정 요소를 클릭합니다. <br />
      추가 시, 브라우저 화면에서 대상 요소를 선택하여 액션 추가를 완료합니다.
    - `Navigate` : 특정 URL로 이동합니다.
    - `Reload` : 현재 페이지를 새로고침합니다.
  - `Add Variable` : 반복되는 텍스트 혹은 함수 실행값을 저장하는 변수를 추가합니다.
