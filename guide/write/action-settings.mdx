---

**액션 우클릭 > \[Settings\]** 버튼을 눌러 액션 상세 옵션을 설정할 수 있습니다.

<img src="/assets/write/action-settings/action-settings.png" />

상세 옵션은 다음과 같습니다.

| 옵션           | 기본값   | 설명                                                                              |
| -------------- | -------- | --------------------------------------------------------------------------------- |
| `Optional Run` | False    | 체크할 경우, **해당 스텝이 실패해도 재생을 계속합니다**.                          |
| `Timeout`      | 3000(ms) | 화면에서 **해당 요소가 나타날 때까지 기다리는 시간의 허용치입니다**.              |
| `Delay`        | 500(ms)  | 액션을 실행하기 전에, **이전 액션의 결과가 업데이트 되도록 기다리는 시간**입니다. |
