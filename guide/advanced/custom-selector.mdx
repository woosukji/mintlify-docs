You can specify elements using [CSS Selectors](https://www.w3schools.com/cssref/css_selectors.php) or [XPath Selectors](https://www.w3schools.com/xml/xpath_syntax.asp).

Click the **element badge following the action's name** to view and set the element's detailed information.

<img width="50%" src="/assets/advanced/custom-selector/custom-selector.png" />

The details are as follows:

| Item                 | Description                                                   |
| -------------------- | ------------------------------------------------------------- |
| `Name`               | Specify the name of the element displayed in the action step. |
| `Custom Selector`    | Specify the element using CSS/XPath selectors.                |
| `Element Screenshot` | View a screenshot image of the element.                       |
