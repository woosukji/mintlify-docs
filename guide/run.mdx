## 1. 테스트 실행하기

**\[프로젝트 화면\]** 에서 실행할 테스트들을 선택하고, 상단의 **\[Run Test\]** 버튼을 클릭합니다.

<Note>
  테스트를 선택하지 않고 실행하면, 프로젝트에 포함된 모든 테스트가 실행됩니다.
</Note>

<img src="/assets/run/run.png" />

테스트 실행 시 사용될 [환경(environment)](/guide/write/env-variable#3)을 선택합니다.

<img src="/assets/run/run-popup.png" />

## 2. 테스트 실행 결과 확인하기

화면 최상단 내비게이션 바에서 **\[Runs\]** 탭을 선택합니다.

테스트 실행 결과 리스트 중 원하는 결과를 선택합니다.

테스트 실행 결과 페이지에서, 해당 실행에 포함된 테스트 별 실행 결과를 확인할 수 있습니다.

<img src="/assets/run/run-result.png" />

각 테스트를 클릭하면, 해당 테스트의 액션 로그를 확인할 수 있습니다.

화면 좌측 상단의 햄버거 버튼을 클릭하여, 테스트 폴더별로 결과를 선택할 수 있습니다.

테스트 결과는 영상 녹화로 확인하실 수 있으며, 좌측의 각 액션을 클릭하여 해당 시점의 화면을 확인할 수 있습니다.
