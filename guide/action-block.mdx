테스트 시나리오에서 반복적으로 사용되는 일련의 액션들을 묶어서 액션블록으로 저장하고, 한 번에 추가할 수 있습니다.

## 1. 액션블록 생성하기

액션블록으로 묶고 싶은 범위를 `Shift` 키를 눌러 선택합니다.

**액션 범위 우클릭 > \[Create Action Block\]** 버튼을 눌러 액션블록을 생성합니다.

액션블록의 이름을 설정합니다.

<Note>
  액션블록의 이름 또는 내부 액션을 수정한 뒤에는, <br />
  액션블록 상단의 **[Update]** 버튼을 눌러 변경사항을 다른 테스트에 반영합니다.
</Note>

<img src="/assets/action-block/create-action-block.png" />

## 2. 액션블록 사용하기

Reliv 에디터 **\[Test Steps\]** 패널 우상단 **\[+\] 버튼 > \[Add Action Block\]** 버튼을 눌러 액션블록을 추가할 수 있습니다.

<img src="/assets/action-block/use-action-block.png" />

## 3. 액션블록 파라미터

액션블록이 추가되는 위치에 따라서 달라져야 하는 값을, 파라미터로 지정하고 관리할 수 있습니다.

액션블록 내에서 [변수](/guide/write/variable)를 사용하는 것과 유사하지만,

파라미터를 이용하면 액션블록 외부에서 변수 값을 받아와서 사용할 수 있습니다.

### 3-1. 액션블록 파라미터 생성하기

액션블록 상단의 **[+ Add Parameter]** 버튼을 눌러 파라미터를 추가할 수 있습니다.

파라미터의 `Key`는 변수의 이름에, `Value`는 변수의 값에 해당합니다.

파라미터의 값을 설정하는 방법은 [변수 생성하기](/guide/write/variable#1) 와 동일합니다.

<Note>파라미터 값에서, 해당 액션 블록 상위의 변수를 사용할 수 있습니다.</Note>

### 3-2. 액션블록 파라미터 사용하기

텍스트 입력창에서 `{` (중괄호)를 입력하면, 파라미터/변수/환경변수/함수 목록이 나타납니다.

클릭하여 파라미터를 삽입할 수 있습니다.
